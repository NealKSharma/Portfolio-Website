BrewMaster – Full-Stack E-Commerce Platform

A full-stack e-commerce web application developed as BrewMaster, a simulated coffee retail store. Designed for scalability and security, it implements a domain-independent retail management system supporting user accounts, product inventory, and order processing. Built using ASP.NET Core MVC, C#, and SQL Server, it follows the MVC architecture pattern with a centralized Database Helper for clean data handling, AJAX-driven UI updates, and robust authentication mechanisms.

Key Features & Contributions:

  • Implemented secure user authentication with SHA256 + salt hashing, role-based access control, and dynamic profile updates with validation.
  • Developed an interactive admin dashboard to monitor users, orders, and logs in real time.
  • Built AJAX-based product management with live table updates, binary image uploads, and autofill editing functionality for streamlined admin operations.
  • Designed a responsive shopping cart and checkout system with SQL data sync, notifications, and automatic stock handling.
  • Engineered detailed audit trails and error logging using structured SQL tables for system transparency.
  • Deployed on Microsoft Azure with Function Apps and Key Vault integration for secure, cloud-ready scalability.

Network Attached Storage (NAS)

Personal Server Infrastructure & Automation System

A self-hosted Network Attached Storage (NAS) and personal server setup running on a Raspberry Pi 5, designed for automated backups, remote file access, and secure networking. Built with OpenMediaVault on Debian Linux, it provides shared network folders accessible from multiple devices, with backup automation and security tools that emulate lightweight DevOps and cloud practices in a home-lab environment.

Key Features & Implementations:

  • Configured OpenMediaVault on Debian Linux to manage file sharing, system health, and network drives accessible from Windows and Linux PCs.
  • Developed bash automation scripts for daily and remote server backups, including rotation management and Wake-on-LAN triggers for power-efficient operations.
  • Integrated OpenVPN for encrypted remote access and deployed Docker-based services for modular application hosting and system isolation.
  • Set up a local web hosting environment for tools and web apps, with real-time system monitoring via a custom dashboard.
  • Hardened the server with custom firewall rules and network policies to ensure secure and stable 24/7 uptime.
